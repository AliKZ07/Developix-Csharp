معرفی کتابخانه های کاربردی

QWxp

در جدول زیر برخی از کتابخانه های کاربردی به همراه لینک گیتهاب  نام برده شده اند

 

توضیحات نام کتابخانه
تبدیل رشته ها به متون قابل فهم توسط انسان،کار با استرینگHumanizer
MockingNSubstitute
 Functional Programming کردن به روش  returnOneOf
faker.js تولید دیتای فیک .ساخته شده بر اساس Bogus
 به صورت یک خطی Value Object ساخت ValueOf
Youtube ساخت رشته های تصادفی مشابه بهHashids.Net
برای تلگرام Client Api ساخت ربات هایWTelegramClient
برای تلگرام Bot Api ساخت ربات هایTelegram.Bot
 ساخت و تنظیم سیاست های مختلف برای مدیریت خطاPolly
MockingMoq4
های بهتر و دقیق تر  StacktraceBen.Demystifier
asp.net core برای  Diagnostics ابزار رفع مشگل وHealthChecks
TDD برای راحتی کار در  Fluent کردن به صورت AssertFluentAssertions
Background زمان بندی کار ها و انجام پردازش ها درHangfire
بنچمارک گرفتن از کدها در سی شارپBenchmarkDotNet
Cli فریمورکی برای توسعه اپلیکیشن های مبتنی برSharprompt

 

اگه شما هم کتابخانه ای میشناسید میتونید مشارکت کنید
